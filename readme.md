GAME IN SPAAAACE!
====

Just playing with Ruby2d (https://ruby2d.com)

#### Install & Run

* `bundle`
* `bundle exec ruby game.rb`

#### Controls

* Rotate Right: `l`
* Rotate Left: `j`
* Move Forward: `i`
* Move Backward: `(you don't have the rockets for that)`
* Fire laser: `space`
